Los parrafos se ponen con p entre "<>" Esto aplica para cada tipo de etiqueta.

Se puede poner todos los parrafos que se desee

Para cortar el texto en parrafos muy largos se puede usar br.
Esto incluye un break o enter en el parrafo y obliga el resto a ir en una nueva linea.

Asi se ponen cursivas
Asi se ponen negitas
Se pueden poner las dos al mismo tiempo tambien

Listas

Las listas se ponen entre ul con su respectivo <>

    ul es para listas sin orden o vinetas. ol da listas con numeros

    Cada elemento de la lista se marca con li

  • Lista
  • Lista 2
  • Lista 3
  • Lista 4

    Aca hay una lista numerada.

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
